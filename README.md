**Career Up** is a high-fidelity, production-ready Student Management System designed for modern EdTech platforms. Built with a focus on "Elite Dark" aesthetics and seamless user experience, it bridges the gap between student enrollment and professional career growth.

---

## ✨ Key Features

### 🎓 Student & Academic Management
- **Centralized Dashboard**: High-contrast, premium interface for students to track progress.
- **Personalized Profiles**: Manage student details, profile pictures, and professional resumes.
- **Course Lifecycle**: Specialized paths for Python Full Stack, Java, MERN, and AI/ML.

### 💳 Financial & Enrollment Hub
- **Transparent Fee Tracking**: Clear breakdown of course fees, paid amounts, and remaining balances.
- **Fast-Track Enrollment**: Integrated request forms with direct advisor triggers (WhatsApp/Phone).
- **Automated Calculations**: Real-time balance updates for student accounts.

### 🤖 CORTEX AI Assistance
- **Intelligent Support Hub**: Integrated AI chatbot (CORTEX) for instant student queries.
- **Quick-Chips Navigation**: rapid access to fee info, mentorship, and enrollment routes.

### 🎨 Premium "Elite" UI/UX
- **Glassmorphism Design**: Modern, translucent UI elements with ambient background effects.
- **Dynamic Animations**: Powered by GSAP for smooth transitions and interactive cursors.
- **Mobile Responsive**: Fully optimized for any device.

---

## 🛠️ Tech Stack

- **Backend**: Python 3.x, Django Web Framework
- **Frontend**: HTML5, Vanilla CSS3, Bootstrap 5
- **Interactivity**: JavaScript (ES6+), GSAP (GreenSock Animation Platform)
- **Database**: SQLite (Development) / PostgreSQL (Production Ready)
- **Icons & Fonts**: Font Awesome 6.5, Google Fonts (Outfit & Inter)

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/YourUsername/StuMng_Proj.git
cd StuMng_Proj
```

### 2. Set Up Virtual Environment
```bash
python -m venv venv
# On Windows:
.\venv\Scripts\activate
# On macOS/Linux:
source venv/bin/activate
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```

### 4. Database Setup
```bash
python manage.py makemigrations
python manage.py migrate
```

### 5. Run Development Server
```bash
python manage.py runserver
```
Access the platform at `http://127.0.0.1:8000/`
